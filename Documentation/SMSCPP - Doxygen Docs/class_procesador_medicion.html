Medicion * ProcesadorMedicion::procesarMedicion ( SMS *  mensaje  ) 

Con el mensaje recibido construye una medición y valida sus datos

Procesa el contenido de un mensaje de texto para poder obtener la representación en objeto de la medición registrada

Se encarga de validar si el contenido del mensaje recibido corresponde a una medición. Caso contrario retorna objeto nulo.

Parameters:
SMS Mensaje de texto con contenido recibido
Returns:
Medicion Objeto con datos de medición u objeto NULO si la medición no es válida o existe algún problema procesandola.
