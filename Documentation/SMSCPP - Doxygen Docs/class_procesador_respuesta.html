void ProcesadorRespuesta::procesarRespuesta (  ) 

Procesa una respuesta escuchando nuevas inserciones. Envía respuesta al paciente

Obtiene una respuesta desde la tabla de backup. Es llamado luego de una notificación por inserción desde la base de datos. Obtiene el contenido de la respuesta, obtiene el identificador del paciente al que se le está respondiendo. Por último elimina la respuesta de la tabla de backup para que no sea reprocesada por otra llamada a este método.
